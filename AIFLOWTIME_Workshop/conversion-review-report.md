## 2. CTA FREQUENCY AND PLACEMENT ⚠️ NEEDS IMPROVEMENT

### Current CTA Distribution:
| Location | CTA Text | Type |
|----------|----------|------|
| Header Nav | 立即預約 | Primary |
| Hero | 立即預約免費試堂 | Primary |
| Workshop 0 | 立即報名 | Primary |
| Workshop A | 了解詳情 | Outline |
| Workshop B | 立即報名 | Primary |
| Workshop C | 開始自我成長之旅 | Outline |
| Workshop D | 了解詳情 | Outline |
| Final CTA | 立即預約免費諮詢 | Primary |

### Issues Identified:
1. **Inconsistent CTA wording** - Creates confusion:
   - Hero: "免費試堂" (free trial class)
   - Final CTA: "免費諮詢" (free consultation)
   - These are DIFFERENT offers

2. **Mixed intent signals**:
   - Some CTAs say "報名" (register/enroll) - implies commitment
   - Some say "了解詳情" (learn more) - implies browsing
   - No clear distinction between high-intent and low-intent actions

3. **Missing mid-page CTA**:
   - No CTA between Pain Points and Workshops sections
   - Users who resonate with pain points have to scroll to find next step

### Recommendations:
- **Standardize primary CTA**: Use consistent wording throughout (suggest "立即預約免費諮詢")
- **Add CTA after Pain Points section**: "想擺脫OT？立即了解解決方案 →"
- **Use consistent button hierarchy**:
  - Primary button (filled): For main conversion action
  - Secondary button (outline): For secondary actions only
- **Current outline buttons should either be primary OR removed** - they dilute conversion focus

## 3. URGENCY/SCARCITY ❌ WEAK

### Current State:
- Only ONE urgency element: "名額有限，額滿即止" at the very bottom
- No urgency in hero section
- No specific numbers or deadlines
- No visual urgency indicators

### Issues:
- Generic "limited spots" feels like an afterthought
- No FOMO created above the fold
- Urgency appears only after user has scrolled through entire page
- Not believable - no specific numbers or time constraints

### Recommendations (Natural, Not Pushy):
1. **Add specific numbers**:
   - "每月只開2班，每班限8人"
   - "下月課程尚餘3個名額"

2. **Add time-based urgency**:
   - "下月課程報名截止日期：X月X日"
   - "早鳥優惠至X月X日"

3. **Add urgency to hero**:
   - Below CTA button: "下月班尚餘少量名額"

4. **Add visual urgency indicators**:
   - Small badge on workshop cards: "熱門" or "快滿"
   - Subtle pulse animation on CTA button
