Browser Support

The glassmorphism effect uses backdrop-filter which is supported in all modern browsers. Fallbacks are provided for older browsers.

✅ Full Support

  • Chrome 76+
  • Safari 9+
  • Firefox 103+
  • Edge 79+
  • iOS Safari 9+
  • Chrome Android 76+

⚠️ Fallback Applied

  • Internet Explorer (solid background)
  • Older Firefox (increased opacity)
  • Legacy browsers (opaque card)

Fallbacks use @supports for graceful degradation.

♿ Accessibility

  • prefers-reduced-motion
  • prefers-reduced-transparency
  • prefers-contrast: high
  • prefers-color-scheme: dark
  • Focus visible states
