# Traditional Chinese Copywriting Style Guide
## Educational Products | Hong Kong & Taiwan Markets
### For Beginner AI Workshop Landing Page

---

## 1. TONE & STYLE GUIDELINES

### Core Tone Principles

| Principle | Description | Example |
|-----------|-------------|---------|
| **Warm (溫暖)** | Make readers feel welcomed and understood | 「我們懂你的擔心」not 「注意事項」 |
| **Confident (自信)** | Show expertise without arrogance | 「我們會帶你一步步完成」not 「我們是最專業的」 |
| **Approachable (親切)** | Use conversational, everyday language | 「一起來試試看」not 「請進行操作」 |

### Tone Do's and Don'ts

**✅ DO:**
- Use "你" (you) to create personal connection
- Include reassuring phrases like "別擔心" / "不用怕"
- Write as if speaking to a friend over coffee
- Use active, encouraging verbs

**❌ DON'T:**
- Use overly formal business language (e.g., "敬請", "貴賓")
- Sound condescending (e.g., "這很簡單，你一定會")
- Use imperative commands without context
- Over-promise with hype language

### Good vs Bad Tone Examples

| Section | ❌ BAD (Too Formal/Cold) | ✅ GOOD (Warm & Approachable) |
|---------|-------------------------|------------------------------|
| Header | AI人工智能基礎課程報名 | 2.5小時，讓AI開始幫你工作 |
| Introduction | 本課程專為零基礎學員設計 | 完全沒用過AI？沒問題，我們從頭開始 |
| Reassurance | 無需擔憂學習難度 | 別擔心，你不需要懂任何技術 |
| CTA | 立即提交報名申請 | 我要預留我的座位 |

## 2. AVOIDING TECHNICAL JARGON

### English Tech Terms to AVOID

| ❌ AVOID (English) | ❌ AVOID (Direct Translation) | ✅ USE INSTEAD (Plain Chinese) |
|-------------------|------------------------------|-------------------------------|
| Prompt Engineering | 提示詞工程 | 跟AI說話的技巧 / 怎麼問AI才會聽懂 |
| Prompt | 提示詞 | 指令 / 你想跟AI說的話 |
| Few-Shot Learning | 少樣本學習 | 給AI幾個例子，讓它學會你的風格 |
| LLM (Large Language Model) | 大型語言模型 | AI的大腦 / 聰明的AI系統 |
| API | 應用程式介面 | AI的連接方式 (or skip entirely) |
| Token | 符元/代幣 | AI理解文字的小單位 |
| Hallucination | 幻覺 | AI說錯話 / AI給出錯誤答案 |
| Fine-tuning | 微調 | 讓AI更懂你的需求 |
| Parameters | 參數 | AI的設定 (or skip entirely) |
| Neural Network | 神經網路 | AI的學習系統 (or skip entirely) |
| Algorithm | 演算法 | AI的運作方式 (or skip entirely) |
| Machine Learning | 機器學習 | AI自己學習的能力 |
| Deep Learning | 深度學習 | 進階的AI學習 (or skip entirely) |
| Workflow | 工作流程 | 做事的步驟 / 一連串操作 |
| Integration | 整合 | 把不同工具連在一起 |
| Automation | 自動化 | 讓AI自動幫你完成 |
| Output | 輸出 | AI給你的答案 / 結果 |
| Input | 輸入 | 你給AI的內容 / 告訴AI的事 |

### How to Describe Technical Benefits in Plain Chinese

**Instead of:** "Learn prompt engineering techniques"
**Say:** "學會怎麼跟AI溝通，讓它聽懂你想要的"

**Instead of:** "Master LLM fundamentals"
**Say:** "了解AI是怎麼理解你說的話"

**Instead of:** "Build automated workflows"
**Say:** "設計讓AI自動幫你做事的步驟"

**Instead of:** "Optimize token usage"
**Say:** "讓AI更快給你想要的答案"

## 3. BEFORE/AFTER FRAMING

### Describing "Before" States (Without Making People Feel Bad)

**❌ AVOID (Shaming):**
- "你還在用老方法做事嗎？"
- "不懂AI的人已經被淘汰了"
- "你浪費了多少時間在重複工作上？"

**✅ USE INSTEAD (Empathetic):**
- "每天被重複的工作佔據，沒時間做真正重要的事？"
- "聽說AI很有用，但不知道從哪裡開始？"
- "試過AI工具，但覺得它聽不懂你的需求？"
- "想學AI，但擔心自己沒有技術背景？"

### "Before" State Phrases That Build Connection

| Pain Point | Empathetic Framing |
|------------|-------------------|
| Overwhelmed by work | "事情永遠做不完，時間總是不夠用" |
| Tried AI but confused | "打開ChatGPT，不知道要問什麼" |
| Fear of being left behind | "看到別人用AI效率倍增，自己卻不知道怎麼開始" |
| Don't know where to start | "網路上AI資訊太多，越看越混亂" |
| Think AI is too technical | "一直以為AI是工程師才能用的工具" |

### "After" States That Feel Achievable

**❌ AVOID (Over-promising):**
- "成為AI專家"
- "掌握所有AI工具"
- "工作效率提升1000%"

**✅ USE INSTEAD (Realistic & Inspiring):**
- "能夠用AI幫你完成日常工作的30%"
- "知道怎麼問AI，才能得到有用的答案"
- "省下每天2小時的重複性工作"
- "有信心繼續探索更多AI應用"

### Before/After Copy Examples

**Example 1: Time Saving**
- Before: "每天花3小時回覆類似的郵件"
- After: "用AI幫你草擬回覆，你只需要確認發送"

**Example 2: Overcoming Fear**
- Before: "看到AI新聞就焦慮，覺得自己會被淘汰"
- After: "把AI當成助手，專注在更有創意的工作上"

**Example 3: Getting Started**
- Before: "打開AI工具，盯著空白畫面不知道說什麼"
- After: "有明確的步驟，知道怎麼引導AI幫你完成任務"

## 4. CTA (CALL TO ACTION) COPY

### Effective CTA Phrases in Traditional Chinese

#### Primary CTAs (High Commitment)

| Standard | Alternative Variations |
|----------|----------------------|
| 我要報名 | 預留我的座位 / 鎖定早鳥優惠 |
| 立即報名 | 開始我的AI學習之旅 |
| 免費報名 | 免費搶先體驗 |

#### Secondary CTAs (Low Commitment)

| Purpose | CTA Copy |
|---------|----------|
| Learn more | 了解課程詳情 / 看看會學到什麼 |
| Save for later | 先加入行事曆 / 傳送到我的信箱 |
| Share | 分享給需要的朋友 |
| Questions | 有問題？先聊聊 |

### CTA Friction-Reducing Phrases

Add these near CTAs to reduce anxiety:

| Concern | Reassuring Phrase |
|---------|------------------|
| Time commitment | "只需要2.5小時" / "一個下午就能學會" |
| Money concern | "完全免費" / "零成本體驗" |
| Skill requirement | "不需要任何背景" / "從零開始也沒問題" |
| Fear of spam | "我們不會寄垃圾信" / "隨時可以取消" |
| Will it work for me? | "超過XXX人都學會了" / "適合完全新手" |

### CTA Button Copy Examples

| ❌ WEAK | ✅ STRONG |
|---------|----------|
| 提交 | 我要預留座位 |
| 送出 | 免費報名體驗 |
| 註冊 | 開始學習AI |
| 報名 | 2.5小時後，讓AI幫我工作 |

## 5. HONG KONG VS TAIWAN DIFFERENCES

### Key Terminology Differences

| English | Taiwan (台灣) | Hong Kong (香港) |
|---------|--------------|------------------|
| Software | 軟體 | 軟件 |
| Hardware | 硬體 | 硬件 |
| Internet | 網路 / 網際網路 | 互聯網 |
| Network | 網路 | 網絡 |
| Laptop | 筆記型電腦 | 手提電腦 |
| Smartphone | 智慧型手機 | 智能手機 |
| Resolution | 解析度 | 解像度 |
| Social Media | 社群媒體 | 社交媒體 |
| Blog | 部落格 | 網誌 |
| Forum | 討論區 | 論壇 |
| File | 檔案 | 檔案 |
| Folder | 資料夾 | 文件夾 |
| Save | 儲存 | 儲存 |
| Copy | 複製 | 複製 |
| Paste | 貼上 | 貼上 |

### Tone Differences

| Aspect | Taiwan Preference | Hong Kong Preference |
|--------|------------------|---------------------|
| Formality | Slightly more formal | More casual, direct |
| English mixing | Less English in copy | More comfortable with English terms |
| Visual style | Warm, friendly colors | Modern, sleek design |
| Sentence structure | Standard Mandarin | May include Cantonese influences in casual contexts |

### Writing Style Recommendations

**For Taiwan Market:**
- Use 軟體, 硬體, 網路
- Slightly more polite/formal tone acceptable
- Avoid excessive English terms
- Use 裡 (not 裏)

**For Hong Kong Market:**
- Use 軟件, 硬件, 互聯網
- Can be more casual and direct
- English terms more acceptable (e.g., "AI工具" is fine)
- Use 裏 (not 裡)

**Universal (Works for Both):**
- 檔案, 複製, 貼上, 儲存
- Warm, encouraging tone
- Focus on benefits over features

## 6. TRUST-BUILDING PHRASES

### Words That Build Credibility

| Category | Phrases |
|----------|---------|
| Experience | "我們已經教過XXX位學員" / "超過X年的教學經驗" |
| Results | "學員平均XXX的進步" / "95%的學員推薦這門課" |
| Approachability | "沒有笨問題" / "隨時可以發問" |
| Guarantee | "不滿意全額退費" / "免費重修" |
| Transparency | "課程內容完全公開" / "沒有隱藏費用" |

### Anxiety-Reducing Phrases

| Concern | Reassuring Phrase |
|---------|------------------|
| "I'm not technical" | "這堂課專為非技術背景設計" / "不需要寫任何程式" |
| "I might not keep up" | "課後提供錄影回放" / "可以照自己的步調學習" |
| "What if I don't understand" | "現場有助教協助" / "課後有專屬社群可以發問" |
| "Is this too basic/advanced" | "我們會確認這堂課適合你" / "課前有程度評估" |

### Social Proof Phrases

- "已經有XXX人報名"
- "學員推薦率XX%"
- "畢業學員現在都在..."
- "看看學員怎麼說"
- "加入超過XXX人的學習社群"

## 7. COMMON TRANSLATION PITFALLS

### Pitfall 1: Direct Translation from English Marketing Copy

**❌ BAD (Direct translation):**
"Unlock the power of AI" → "解鎖AI的力量"

**✅ GOOD (Natural Chinese):**
"讓AI開始為你工作" / "發現AI能怎麼幫你"

### Pitfall 2: Overusing English Terms

**❌ BAD:**
"在這個workshop，你會學到Prompt的技巧"

**✅ GOOD:**
"在這堂課，你會學會怎麼跟AI溝通"

### Pitfall 3: Being Too Formal

**❌ BAD:**
"敬請報名參加本課程"

**✅ GOOD:**
"一起來學習吧！"

### Pitfall 4: Over-promising

**❌ BAD:**
"成為AI大師" / "徹底改變你的人生"

**✅ GOOD:**
"開始用AI簡化你的工作" / "省下每天寶貴的時間"

### Pitfall 5: Ignoring Cultural Context

**❌ BAD:**
"You deserve this" → "你值得擁有這個"

**✅ GOOD:**
"給自己一個學習的機會" / "投資自己，從這堂課開始"

## 8. RECOMMENDED PHRASINGS FOR WORKSHOP BENEFITS

### Instead of Technical Features, Focus on Benefits

| Technical Feature | Benefit-Focused Phrasing |
|-------------------|-------------------------|
| Learn ChatGPT | 學會讓AI幫你寫文案、回郵件、整理資料 |
| Master prompting | 知道怎麼問AI，才能得到有用的答案 |
| Build AI workflows | 設計讓AI自動幫你完成重複工作 |
| Understand LLMs | 了解AI是怎麼理解你的需求 |
| Use AI tools | 把AI變成你的日常助手 |

### Workshop Benefit Statements

**Headline Options:**
- "2.5小時，讓AI開始幫你工作"
- "從零開始，學會用AI省下每天2小時"
- "不懂技術也能駕馭AI"
- "AI新手的第一堂課"

**Subheadline Options:**
- "不需要寫程式，不需要懂技術，只需要帶著好奇的心"
- "超過XXX位新手，從這裡開始他們的AI之旅"
- "我們會帶你一步步，從完全不懂到能夠實際應用"

**Body Copy Framework:**
1. Acknowledge the struggle (empathy)
2. Present the solution (workshop)
3. Show the transformation (after state)
4. Remove objections (reassurance)
5. Call to action (CTA)

## 10. QUICK REFERENCE: WORDS TO USE & AVOID

### Words to USE (Build Connection)

| Word/Phrase | Why It Works |
|-------------|--------------|
| 一起 | Creates sense of community |
| 別擔心 | Reduces anxiety |
| 簡單 | Removes intimidation |
| 一步一步 | Makes learning feel manageable |
| 你會發現 | Creates curiosity |
| 其實 | Makes concepts accessible |
| 就像 | Uses relatable analogies |
| 不用怕 | Addresses fears directly |
| 試試看 | Encourages low-risk action |
| 適合你 | Personalizes the message |

### Words to AVOID (Create Distance)

| Word/Phrase | Why to Avoid |
|-------------|--------------|
| 必須 | Too demanding |
| 應該 | Sounds preachy |
| 學習曲線 | Technical jargon |
| 優化 | Sounds corporate |
| 解決方案 | Too business-like |
| 核心競爭力 | Overused buzzword |
| 顛覆 | Overhyped |
| 革命 | Overpromising |
| 掌握 | Too strong for beginners |
| 精通 | Unrealistic expectation |

## SUMMARY CHECKLIST

Before finalizing your copy, check:

- [ ] Is the tone warm and approachable?
- [ ] Are all technical terms explained or replaced?
- [ ] Does the "before" state feel empathetic, not shaming?
- [ ] Does the "after" state feel achievable?
- [ ] Is the CTA clear and low-friction?
- [ ] Are trust elements included near CTAs?
- [ ] Is the copy appropriate for HK, Taiwan, or both?
- [ ] Are benefits prioritized over features?
- [ ] Is the language conversational, not corporate?
- [ ] Does it sound like a helpful friend, not a salesperson?
