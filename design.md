# AIFlowTime 設計系統

## 設計理念

- 極簡主義：大量留白，乾淨整齊的佈局
- 高級但親民：精緻的視覺細節，但不疏離
- 現代感：使用當代設計語言，符合 AI 科技感
- 信任感：清晰的層次結構，專業的排版

## 色彩系統

**主要色彩**

- 背景色：#FFFFFF（純白底）
- 次要背景：#F8F9FA（極淡灰）
- 品牌主色：#2563EB（青藍色，用於 CTA、重點標示）
- 文字主色：#1F2937（深灰，主要文字）
- 文字次要：#6B7280（中灰，次要資訊）
- 邊框色：#E5E7EB（淡灰，分割線、卡片邊框）

**功能色彩**

- 成功色：#10B981（用於正面提示）
- 警告色：#F59E0B（用於注意事項）

## 字體系統

**主要字體**

- 英文字體：Inter（現代無襯線，清晰易讀）
- 中文字體：Noto Sans TC（繁體中文，風格中性專業）

**字體層級**

- H1 大標題：48px / 3rem，font-weight: 700
- H2 副標題：32px / 2rem，font-weight: 600
- H3 小標題：24px / 1.5rem，font-weight: 600
- 正文：16px / 1rem，font-weight: 400
- 小字：14px / 0.875rem，font-weight: 400

## 視覺效果

**動效原則**

- 克制：動效需有意義，不為炫技
- 流暢：使用 ease-out 緩動函數
- 快速：持續時間 200-400ms
- 微妙：小幅度移動，不搶奪內容風采

**具體動效**

- 文字淡入：opacity 0 → 1，translateY 20px → 0
- 卡片 hover：輕微上浮 4px，陰影加深
- 視差滾動：背景元素移動速度為 0.5x
- 身份輪播：淡入淡出效果，持續 1.2 秒

**交互反饋**

- 按鈕 hover：背景色加深 10%，輕微縮放 1.02x
- 連結 hover：顏色變化，添加底線動畫
- 表單 focus：邊框變為品牌色，添加輕微陰影

## 佈局原則

- 最大寬度：1200px（容器居中）
- 區塊間距：80px（大區塊之間）
- 元素間距：24px（相關元素之間）
- 邊距：左右各 24px（移動端）
- 圓角：8px（按鈕、卡片）
- 陰影：0 4px 6px -1px rgba(0, 0, 0, 0.1)
